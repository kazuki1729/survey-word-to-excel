# survey-word-to-excel

Word 形式のアンケート調査票を、**集計・分析しやすい Excel に自動転記**する Python ツールです。

設問番号・設問文・選択肢を読み取り、**原文のまま**Excelの表に整理します。
プログラムで一括変換するため、**設問番号のズレや転記ミスを防ぎ、大量の設問でも短時間・高精度**で処理できます。

> 📌 アンケートの Word→Excel 転記作業（卒論・研究調査・市場調査など）を想定して制作したツールです。
> サンプルはポートフォリオ用の架空データです。

## ✨ 特徴

- 📄 **Word(.docx) を読み取り**、設問・選択肢を自動で抽出
- 📊 **2種類の形式で Excel 出力**
  - **ワイド形式**：1設問1行＋回答記入欄（そのまま回答入力に使える）
  - **ロング形式**：1選択肢1行（集計・グラフ化など分析に便利）
- 🎯 **原文のまま転記**（略さない）。番号体系も保持
- 🤝 **大量設問に対応**（数百問でも一括処理）
- 🧹 ヘッダー装飾・列幅調整・ウィンドウ枠固定など、**見やすい体裁**で出力

## 📥 対応している設問・選択肢の形式

```
問1. 設問文           ← 「問1.」「Q1)」「Ｑ1：」などに対応
1. 選択肢             ← 「1.」「2)」などの番号
①. 選択肢            ← 丸数字「①②③」
ア. 選択肢            ← カタカナ「ア．イ）」
```

実際の調査票の形式に合わせて、抽出ルール（正規表現）は簡単に調整できます。

## 🚀 使い方

```bash
# 1. ライブラリのインストール
pip install -r requirements.txt

# 2. 変換（入力Word → 出力Excel）
python convert.py 入力.docx 出力.xlsx

# 引数を省略するとサンプルを変換します
python convert.py
```

## 📂 サンプル

`sample/` に、動作確認用のサンプルを同梱しています。

| ファイル | 内容 |
|---|---|
| `sample/sample_survey.docx` | 入力例（架空の意識調査・全12問） |
| `sample/survey.xlsx` | 変換後のExcel（ワイド／ロングの2シート） |

サンプルを再生成する場合：

```bash
python make_sample.py   # サンプルWordを作り直す
python convert.py       # Excelへ変換
```
